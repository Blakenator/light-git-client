# Installation

Light Git Client is available for **macOS**, **Windows**, and **Linux**.

## Download

Head to the [GitHub Releases page](https://github.com/Blakenator/light-git-client/releases) to grab the latest release.

## Distribution Options

### Portable

No installation required. Download, extract, and run.

| Platform | Format |
| -------- | ------ |
| macOS | `.dmg` |
| Windows | `.exe` (portable) |
| Linux | `AppImage` |

- No auto-update (except Linux AppImage)
- Great for trying the app without committing to an install

### Installer (Windows)

- Installs to your system with Start Menu shortcuts
- Includes **auto-update** support — you'll be notified when new versions are available
- Recommended for daily use on Windows

## Prerequisites

- **Git** must be installed and available on your system `PATH`
- The app detects your Git installation automatically, but you can configure a custom path in [Settings](/features/settings)

## Verifying Your Install

After launching the app, open **Settings** to confirm:

1. Your Git version is detected
2. Your user name and email are configured

If Git is not found, check the **Git Config** tab in Settings and set the Git executable path manually.
